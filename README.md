# Сайт адвоката Олександра Осадька

Мінімалістичний сайт: головна сторінка (про мене, послуги, статті, контакти) та окремі сторінки статей.

## Структура

- `index.html` — головна сторінка
- `articles/*.html` — статті
- `css/style.css` — стилі
- `assets/logo.svg` / `assets/logo-dark.svg` — логотип (світла та темна версії)
- `assets/photo.jpg` — **фотокартка адвоката (потрібно додати файл вручну)**
- `.github/workflows/pages.yml` — автодеплой на GitHub Pages при пуші в `main`

Сайт опубліковано: https://alexosadko.github.io/web_site/

## Що потрібно доробити

1. Покласти фото в `assets/photo.jpg` (портретна орієнтація, приблизно 3:4).
2. Замінити номер телефону-заглушку `+38 (000) 000-00-00` в `index.html` (секція «Контакти») на справжній.
3. За потреби відкоригувати ім'я, перелік послуг і текст «Про мене».

## Локальний перегляд

Відкрити `index.html` у браузері або запустити простий сервер:

```bash
python3 -m http.server 8000
```

і перейти на http://localhost:8000
